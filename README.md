# SpotFuck-Web

SpotFuck-Web is an open-source **private** music player website. Each user has a private library, and tracks are only accessible while authenticated.

Main Page: https://spotfuck.netlify.app/

## Features
- User authentication
- Private per-user music library
- Upload audio tracks
- Authenticated streaming (with seek via Range)
- Simple web player UI

## Requirements
- Node.js
- A database (e.g., PostgreSQL)
- Audio storage (e.g., S3/MinIO or local for development)

## Setup
1. Clone the repo
2. Copy `.env.example` to `.env`
3. Start the backend
4. Start the frontend
5. Open the web app and sign in
